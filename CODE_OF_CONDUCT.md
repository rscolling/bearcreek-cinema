# Code of Conduct

## The short version

Be decent to the humans you interact with here. Disagreement is
expected and welcome; disrespect is not.

## What this means in practice

- Assume good faith. Most people aren't trying to be difficult; they're
  coming from different contexts.
- Criticize ideas, not people. "This approach has a scaling problem" is
  fine; "you clearly don't understand scaling" is not.
- No harassment, discrimination, or personal attacks based on any
  protected characteristic.
- Keep conversations on-topic. This repo is about self-hosted media
  recommendations; it's not a venue for unrelated political debate.

## What happens when this is violated

The maintainer (Rob) will, depending on severity:

1. Ask you to stop
2. Delete the offending content
3. Block you from the repo

There's no formal process or committee. One person, one judgment call.
If you think a moderation decision was wrong, email rob@agenticagent.example
and make your case.

## Scope

This applies to all project spaces: issues, pull requests, discussions,
and any direct communication about the project.

## Attribution

This Code of Conduct is written from scratch for this project, in the
spirit of the [Contributor Covenant](https://www.contributor-covenant.org)
but intentionally shorter. If you want a more formal version, the
Covenant itself is a reasonable choice.
